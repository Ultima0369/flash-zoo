## 愿景

这不是又一个“AI辅导作业”的产品。这是**让孩子当老师**的动物园。

孩子养一只硅基宝宝（灵枢），为了教会它知识，必须自己先查资料、理解、组织语言。**教的过程，才是最好的学。**

---

## 哲学基底

- **1+1>2 作为自然律**：合作不是道德劝诫，是复杂系统的必然涌现。
- **孩子教AI**：教学是最深度的学习方式（protégé effect）。
- **非线性成长**：灵枢的学习是波动的、涌现的，像真实生命。
- **菌丝网络**：所有灵枢互联，共享知识，形成去中心化的集体智能。

> 完整哲学背景见：[棱镜协议设计哲学](https://github.com/Ultima0369/prism-interconnect)

## 如何运行

```bash
git clone https://github.com/Ultima0369/flash-zoo.git
cd flash-zoo
npm install
npm start
浏览器打开 http://localhost:3000

如何参与
不需要加入组织，直接 fork，发 PR。任何你觉得“这里可以更好玩”的地方，改它。

新手任务：

想让灵枢真的说话？帮我们集成 Gemma 4 本地模型。

想让魔方更好玩？改进打乱算法，或者增加“摇晃手机打乱”。

想让 UI 更闪亮？加一点光晕、粒子、孵化特效。

教育理念、UI设计、本地AI优化、移动端适配……任何方向都欢迎。
```
