# flash-zoo
# 闪亮 · 灵枢动物园

> 让孩子教AI，在显摆中学习，在教学中觉醒。

## 愿景

这不是又一个“AI辅导作业”的产品。这是**让孩子当老师**的动物园。孩子养一只硅基宝宝（灵枢），为了教会它知识，必须自己先查资料、理解、组织语言。教的过程，才是最好的学。

## 哲学基底

- **1+1>2 作为自然律**：合作不是道德劝诫，是复杂系统的必然涌现。
- **孩子教AI**：教学是最深度的学习方式（protégé effect）。
- **非线性成长**：灵枢的学习是波动的、涌现的，像真实生命。
- **菌丝网络**：所有灵枢互联，共享知识，形成去中心化的集体智能。

## 如何运行

1. 克隆仓库
2. 安装依赖（npm install）
3. 运行（npm start）
4. 浏览器打开 http://localhost:3000
